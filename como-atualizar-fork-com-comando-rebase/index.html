Atualizando fork com rebase

Guia com os comandos necessários para atualizar o fork com comando rebase.

  1. Set o upstream caso já não o tenha setado 
git remote add upstream https://github.com/original-repo/goes-here.git
  1. Pegue todas as modificações e branches do upstream 
git fetch upstream
  1. Com rebase reescrevemos a nossa master com o que foi baixado no passo anterior
git rebase upstream/master
  1. Agora é pushar as mudanças do clone local para o repositório fork no Git
git push origin master
  1. Na página do repositório fork no github será possível ver a informação de que agora está seu estado é de even (in sync), ou seja, sincronizado com o repositório de origem.
